# Landing - Impresión 3D Pro

Proyecto demo: Landing page creada con Vite + React + TypeScript + TailwindCSS.

Colores: fondo blanco, textos negros y acentos en celeste (variable `--brand`). Estilo serio y profesional, responsive.

Accesibilidad y UX: incluye enlace de salto (`skip-link`), mejoras en focus states y soporte para navegación por teclado. También dispone de un menú móvil accesible y formularios con atributos ARIA para facilitar su uso.

Tipografía: usa **Inter** (importada desde Google Fonts) para un aspecto moderno y profesional. Se añadió un `favicon.svg` en `src/assets` como icono del sitio.

## Requisitos
- Node.js 18+ y npm / pnpm / yarn

## Instalación
1. Instalar dependencias

```bash
npm install
```

2. Ejecutar en desarrollo

```bash
npm run dev
```

3. Construir para producción

```bash
npm run build
npm run preview
```

Si quieres podemos integrar un form real, agregar más productos o ajustar colores y tipografías.
